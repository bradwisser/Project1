As the title suggests, this is a page dedicated to the first project for the Epicodus evening class. The class is wrapping up a section that introduces students to Git, Cascading Styling Sheets (CSS), and HTML. It would take the students from having very little knowledge of building websites (and for most working with Macs and Bash) to being able build responsive and somewhat basic websites using HTML and CSS. For me, I had already been working ahead on Ruby on Rails and JavaScript, so a lot of the work we covered in the section was repetitive; however, the classes and partners forced me to practice--which I'm grateful for.

The requirements for this project are to build a basic website that includes the information and elements/practices below:

  • Your name
  • An image of you or something that represents who you are
  • A list of projects you have created with active links to them including a description
  • An "About Me" section including background information and the reason for enrolling at Epicodus
  • Create a gh-pages branch so that this site will be live on GitHub Pages--including a README for the project

Our page should include the below elements and practices:

  1. HTML tags: p, h, ul, ol, li, em, strong, a, img, div, span
  2. Bootstrap classes
  3. A custom-made stylesheet that uses cascading, the box model, and floats
  4. Proper indentation and spacing of HTML/CSS
